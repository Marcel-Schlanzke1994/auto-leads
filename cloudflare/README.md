# Cloudflare Optional Integration (Reference)

Dieser Ordner enthält ausschließlich optionale Referenzdateien für spätere Cloudflare-Prototypen.

- Keine produktive Pflichtabhängigkeit für den lokalen Betrieb.
- Keine echten Secrets oder IDs eintragen.
- Deployments erst nach separater Sicherheits- und Architekturfreigabe.
